## 🔬 四层过滤系统

```
┌─────────────────────────────────────────────────────────────┐
│  第一层 · EMA 趋势方向                                        │
│  EMA21 > EMA55 > EMA200（多头）                              │
│  EMA21 < EMA55 < EMA200（空头）                              │
│  → 判断：趋势方向是否存在？                                    │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  第二层 · ADX 趋势持续性确认 ⭐ V2.1 更新                     │
│  ADX >= 30 → 趋势确认，允许入场                               │
│  ADX <  30 → 跳过（震荡市/趋势形成中）                        │
│  ADX 30-35 → 仓位 5%（趋势刚确认）                           │
│  ADX 35-45 → 仓位 10%（趋势确立）                            │
│  ADX >= 45 → 仓位 5%（极强趋势，缩仓保护）                    │
│  ADX >= 40 → 止损扩大至 2.5×ATR（V2.2）                      │
│  → 判断：趋势是否真的在持续？                                 │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  第三层 · 市场结构（入场时机）                                 │
│  看多：HL回调（SL1>SL2）或 BOS突破（>SH1）                   │
│  看空：LH反弹（SH1<SH2）或 BOS跌破（<SL1）                   │
│  → 判断：入场时机是否合适？                                   │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  第四层 · ATR 动量确认                                       │
│  0.7x <= ATR倍率 <= 1.8x → 正常，允许入场                    │
│  ATR倍率 < 0.7x → 跳过（市场睡着了）                         │
│  ATR倍率 > 1.8x → 跳过（波动过大）                           │
│  资金费率 |fee| > 0.1% → 跳过（方向已过度拥挤）              │
│  → 判断：市场是否有足够动量？                                 │
└──────────────────────────┬──────────────────────────────────┘
                           ▼
                    ✅ 四层全过 → 入场
```

## 🔄 持仓监控规则（V2.2 更新）

持仓期间每次检查循环执行：

1. **ADX 持续下降？** ADX 从 ≥ 35 回落至 < 25 → 立即平仓 ⭐ V2.2（收紧）
2. **EMA 反转？** EMA21 穿越 EMA55 → 立即平仓
3. **EMA 距离收窄？** EMA21 与 EMA55 距离 < 0.5×ATR → 减仓 50% ⭐ V2.2
4. **连续止损保护？** 最近 3 笔全部亏损 → 暂停 24 小时 ⭐ V2.2
5. **止盈/止损单丢失？** → 立即补挂

### V2.2 新增保护机制

**EMA 距离监控（趋势衰竭早期预警）**
```
EMA 距离 = |EMA21 - EMA55|
若 EMA 距离 < 0.5 × ATR14：
  → 短期与中期趋势趋于一致，momentum 减弱
  → 减仓 50%，保留部分仓位观察
  → 若距离继续收窄至 < 0.3 × ATR14，全部平仓
```

**连续止损保护（防止回撤扩大）**
```
维护最近 4 笔交易结果队列
若队列中亏损次数 = 4：
  → 触发连续止损保护
  → 暂停新开仓 12 小时
  → 已有持仓正常监控，不受限制
  → 12 小时后重置计数器
```

> ADX 退出保护是 V2.x 相比 V1.0 的重要改进——它比 EMA 反转更早发出退出信号。V2.2 进一步收紧退出条件。

## 🔧 ADX 计算公式（参考实现）

```python
def calculate_adx(highs, lows, closes, period=14):
    # True Range
    tr = pd.concat([
        highs - lows,
        (highs - closes.shift(1)).abs(),
        (lows - closes.shift(1)).abs()
    ], axis=1).max(axis=1)

    # Directional Movement
    up_move  = highs.diff()
    down_move = -lows.diff()
    plus_dm  = ((up_move > down_move) & (up_move > 0)) * up_move
    minus_dm = ((down_move > up_move) & (down_move > 0)) * down_move

    atr     = tr.ewm(alpha=1/period, adjust=False).mean()
    plus_di = 100 * (plus_dm.ewm(alpha=1/period, adjust=False).mean() / atr)
    minus_di= 100 * (minus_dm.ewm(alpha=1/period, adjust=False).mean() / atr)

    dx = 100 * (plus_di - minus_di).abs() / (plus_di + minus_di)
    adx = dx.ewm(alpha=1/period, adjust=False).mean()
    return adx
```
